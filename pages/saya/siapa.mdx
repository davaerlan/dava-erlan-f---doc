# Pengerian HTML
Pengertian HTML menurut Wikipedia:

"*HTML atau HyperText Markup Language merupakan sebuah bahasa markah untuk membuat halaman web.*"

Jadi, HTML itu adalah sebuah bahasa yang menggunakan *markup* atau *penanda* untuk membuat halaman web.

Penanda atau markup ini disebut dengan **tag

Aplikasi text editor yang bisa digunakan untuk belajar penulisan kode HTML.
* Visual Studio Code.
* Notepad++.
* dan lainnya.

# Struktur Dasar HTML
Setiap halaman HTML setidaknya memiliki struktur dasar yang terdiri dari :
* Tag DTD atau DOCTYPE,
* tag html
* tag head
* tag body
